# Leaflet Demo 演示说明

## 功能特性

这是一个基于 Vue 3 + TypeScript + Leaflet 的地图演示项目。

## Quick Start

```bash
pnpm install
pnpm dev
```

## 组件列表

- LeafletMap.vue - 基础地图组件
- MapMeasure_Basic.vue - 基础测距组件
- MapMeasure_Enhanced.vue - 增强版测距组件

## 技术栈

- Vue 3.5
- TypeScript
- Vite
- Leaflet

## 开发分支

本功能在 `feature/leaflet-demo` 分支开发完成。
